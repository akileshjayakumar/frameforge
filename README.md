# FrameForge

**FrameForge** is an innovative AI-powered storytelling game where you collaborate with Gemini AI to craft compelling narratives, then watch them come to life as stunning comic panels and cinematic videos. Built for the [Gemini 3 Hackathon Singapore 2025](https://luma.com/gemini3sgp?tk=whm5Bg).

---

## What is FrameForge?

FrameForge is a turn-based collaborative storytelling experience that pits human creativity against AI intelligence. You and Gemini AI take turns building a story sentence by sentence, creating a unique narrative that neither of you could have written alone. Once your story is complete, FrameForge transforms it into visual artâ€”first as retro sci-fi comic panels, then as a customizable short film.

The game combines the power of **Gemini 3 Flash** for intelligent story generation, **Nano Banana Pro** for comic-style illustrations, and **Veo 3.1** for video creation, all wrapped in a beautiful retro-futuristic interface inspired by 1960s sci-fi aesthetics.

---

## What Does FrameForge Do?

FrameForge provides a complete creative pipeline:

1. **Generates Story Ideas**: AI creates thought-provoking sci-fi story starters
2. **Facilitates Collaborative Writing**: You and AI alternate turns, building a narrative together
3. **Creates Visual Storyboards**: Transforms your completed story into comic panel illustrations
4. **Produces Cinematic Videos**: Generates short films from your story with customizable styles and settings
5. **Tracks Progress**: Shows real-time turn indicators and story progression

---

## How It Works

### Game Flow

#### 1. **Genre & Topic Selection**

- Choose a genre (sci-fi, thriller, fantasy, etc.)
- FrameForge generates 6 unique, thought-provoking story topics using Gemini 3 Flash
- Select the topic that sparks your imagination

#### 2. **Turn-Based Storytelling** (4 turns total)

- **Your Turn**: Choose from 3 AI-generated sentence options that continue the story
- **AI Turn**: Gemini AI automatically adds its own creative sentence
- This alternates for 4 total turns (2 user turns, 2 AI turns)
- Watch your collaborative story unfold in real-time

#### 3. **Storyboard Generation**

- Once the story is complete, FrameForge breaks it into key scenes
- Uses **Nano Banana Pro** to generate retro sci-fi comic-style illustrations
- Creates up to 6 visual panels that bring your narrative to life
- Progress tracking shows image generation status

#### 4. **Video Customization**

- Choose your video style:
  - **Live Action**: Photorealistic cinematic footage
  - **Animation**: Stylized 3D animation (Pixar-quality)
  - **Anime**: Japanese anime style with dynamic angles
  - **Watercolor**: Artistic watercolor animation
- Select duration: 4, 6, or 8 seconds
- Configure aspect ratio (16:9 or 9:16) and resolution (720p or 1080p)

#### 5. **Video Generation**

- Uses **Veo 3.1** to create a cinematic short film from your story
- Reference images from your comic panels guide the video style
- Real-time polling tracks generation progress
- Download your completed video

#### 6. **Final Results**

- View your complete story
- Browse all generated comic panels
- Watch and download your custom video
- Start a new story or continue editing

## Getting Started

### Prerequisites

- Node.js 18 or higher
- npm or yarn package manager
- Gemini API key ([Get one here](https://ai.google.dev/))

### Installation

1. **Clone the repository:**

   ```bash
   git clone <repository-url>
   cd gemini3-2nd
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Create environment file:**
   Create a `.env.local` file in the root directory:

   ```env
   GEMINI_API_KEY=your_api_key_here
   NEXT_PUBLIC_APP_NAME=FrameForge
   ```

4. **Run the development server:**

   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## API Endpoints

### `GET /api/topics`

Generates 6 creative story topics using Gemini 3 Flash.

**Response:**

```json
{
  "topics": [
    { "id": "topic-1", "text": "Story topic text..." },
    ...
  ]
}
```

### `POST /api/story`

Handles story turns and generates content.

**Request Body:**

```json
{
  "turnType": "user-options" | "ai-turn",
  "storySoFar": "Current story text...",
  "genre": "sci-fi",
  "interactionId": "optional-interaction-id"
}
```

**Response:**

- For `user-options`: Returns 3 sentence options
- For `ai-turn`: Returns AI's sentence continuation

### `POST /api/image`

Generates comic-style illustrations from the complete story.

**Request Body:**

```json
{
  "fullStory": "Complete story text...",
  "sceneLabel": "Scene description"
}
```

**Response:**

```json
{
  "imageUrl": "base64-encoded-image-data"
}
```

### `POST /api/video`

Handles video generation, polling, and downloading.

**Actions:**

- `generate`: Start video generation
- `poll`: Check generation status
- `download`: Download completed video

**Request Body (generate):**

```json
{
  "action": "generate",
  "referenceImages": ["base64-image-1", ...],
  "style": "live-action",
  "durationSeconds": 6,
  "aspectRatio": "16:9",
  "resolution": "720p",
  "genre": "sci-fi"
}
```

## Game Mechanics Explained

### Turn System

FrameForge uses a 4-turn system:

1. **Turn 1**: User selects a sentence option
2. **Turn 2**: AI automatically adds a sentence
3. **Turn 3**: User selects another sentence option
4. **Turn 4**: AI adds the final sentence

After 4 turns, the story is complete and ready for visualization.

### Story Generation

- **Topic Generation**: Uses Gemini 3 Flash with specialized prompts to create thought-provoking sci-fi story starters
- **Sentence Options**: AI generates 3 contextually appropriate sentence continuations based on the current story state
- **AI Turns**: Gemini AI uses the Interactions API to maintain story context and generate coherent continuations
- **Genre Consistency**: All generated content respects the selected genre

### Image Generation

- **Story Analysis**: The complete story is analyzed to identify key scenes
- **Panel Creation**: Nano Banana Pro generates retro sci-fi comic-style illustrations
- **Parallel Processing**: Multiple images generated simultaneously for efficiency
- **Style Consistency**: All panels maintain consistent visual style

### Video Generation

- **Reference Images**: Comic panels serve as visual references for video generation
- **Style Application**: Veo 3.1 applies the selected video style (live-action, animation, etc.)
- **Duration Control**: Videos can be 4, 6, or 8 seconds long
- **Aspect Ratios**: Supports both landscape (16:9) and portrait (9:16) formats
- **Resolution Options**: 720p for all durations, 1080p for 8-second videos
- **Async Processing**: Video generation happens asynchronously with polling for status updates
